Why Kazemi

Built around your collection, not someone else’s platform

Kazemi is designed to feel like a real home for your media: clean browsing, polished playback, local folders, downloads, and optional user-managed extensions when you need more flexibility.

Own your library

Keep personal videos, archives, courses, films, or imported catalogs in one interface that feels native on Apple devices.

Designed like an app, not a file browser

Covers, metadata, history, subtitles, and downloads are presented in a clean interface instead of raw folders and filenames.

Flexible when you want it

Kazemi supports local media first, while still allowing user-imported JavaScript extensions for custom sources and extractors.

How it works

A simple flow that stays under your control

Start with local media, expand with user-managed extensions only if you need them.

01

Pick a folder

Select a local folder on your device and let Kazemi organize videos, metadata, covers, and subtitles automatically.

02

Import what you need

Add JavaScript sources or extractors manually when you want additional catalogs or custom playback handling.

03

Watch anywhere

Play online or offline, keep history, manage downloads, and switch subtitle tracks without leaving the app.
